Задание на 1,5 часа в 16:45 репозиторий закрою:

1. Скачать файлы с репозитория, добавить в проект вычисление произведения матриц демонстрацию работы этого метода.
2. Cоздать проект в VS или в QT
3. Написать комментарии для Doxygen и создать pdf документацию.
4. Написать Dockerfile:
	В качестве исходного образа взять 	debian 11.
	Разместить ссылку на репозиторий как метку.
	Установить модуль g++
	Скопировать файлы из репозитория в контейнер в папку MyProj.
	Скомпилировать и запустить программу при старте контейнера.
5. Создать докер образ.
6. Загрузить принтскрины работы с git через powershell или git bash.
7. Загрузить все в репозиторий в ветку с именем <Фамилия>_<номер группы>.

Результат экзамена:
1. Cозданная ветка, например Budylina_201-345
2. В ветке находятся файлы: 
    Dockerfile, 
    main.cpp, 
    matr3d.h, 
    matr3d.cpp 
    файлы проекта *.pro или *.vcxproj, *.sln и *.filters.
    принтскрины работы с git через powershell или git bash
